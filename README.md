# Clustering-Comparative-Study-of-Astronomical-Survey-Data
Analyzed large datasets from multiple sky observatories to detect statistical structure.
Performed comparative analysis to find overlapping patters and anomalies.
WE have used TOPCAT software to do the analysis of the data from different observatory.
List of observatory : eRROSITA, DESI, DES, DES Y3, SDSS.

Project is already complete this repository is just to showcase that project.
